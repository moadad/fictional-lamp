هذه النسخة مخصصة لـ GitHub Pages أو أي استضافة ثابتة.

المميزات:
- Dashboard احترافي
- واجهة جدوليّة للموبايل والكمبيوتر
- رسالة تثبيت تظهر مرة واحدة
- PWA يعمل كتطبيق
- ربط مباشر مع Apps Script API
- تسليم مرن من نهاية الطلبية
- خصم تلقائي من المخزون عبر Apps Script

الخطوات:
1) ارفع هذا المجلد كاملًا إلى GitHub.
2) فعّل GitHub Pages على الفرع الذي رفعت عليه الملفات.
3) افتح الموقع.
4) ضع رابط Apps Script Web App في شاشة الدخول أو الإعدادات.
5) سجّل الدخول.

مهم:
- Apps Script يجب أن يكون منشورًا بصلاحية Anyone.
- إذا عدلت Code.gs يجب عمل Deploy جديد ثم Update.
